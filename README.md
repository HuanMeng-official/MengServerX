### 什么是MengServerX？
MengServerX是 [MengPluginX](https://github.com/HuanMeng-official/MengPluginX) 的续作，相较于MPX，MSX拥有更好的性能，更简洁的代码。

### 为什么要出MengServerX？
原因很简单，MengPluginX堪比屎山的代码属实是让我维护不下去，于是我准备全部推翻重写。

### MengServerX会保持更新吗？
说句现实的，不会，因为我没那个精力，但是我会尽己所能努力维护。
